Atmel AVR Assembler 2

Starting with AVR Studio v. 4.11, AVR Assembler 2 (AVRASM2) is the default AVR assembler. It is a backward compatible replacement for AVRASM with many new features. The old assembler is still distributed with AVR Studio, but no longer maintained (i.e., new parts will no longer be supported by the old assembler).
